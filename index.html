How to Connect

Java Edition

1. Open Minecraft Java Edition.
2. Click "Multiplayer" → "Add Server".
3. Enter minecraft.ezra.is-a.dev as
the server address and connect.

Bedrock Edition (Geyser)

1. Open Minecraft Bedrock Edition.
2. Click "Play" → "Servers" → "Add Server".
3. Enter minecraft.ezra.is-a.dev as
the server address and connect.*

*Note: Keep the port as 19132 for bedrock edition.
